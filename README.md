# Autonomous SQL Repair Agent

A self-healing SQL system that automatically detects query failures,
analyzes root causes using DSPy agents,
generates repair strategies,
and retries execution against a SQLite database.

## Features

- Query execution engine
- Error classification
- Root cause analysis
- Autonomous query repair
- Retry mechanism
- Incident memory

## Tech Stack

- Python
- SQLite
- DSPy
- OpenAI/Gemini- LLM
